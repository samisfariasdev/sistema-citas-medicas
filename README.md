# sistema-citas-medicas
## Instalación y configuración

1. Clona este repositorio.
2. Abre el proyecto en tu IDE favorito (como IntelliJ o Eclipse).
3. Asegúrate de tener Java instalado.
4. Compila y ejecuta `Main.java`.

## Uso del programa

El sistema permite registrar doctores, pacientes y agendar citas médicas. Todos los datos son gestionados en memoria para simular una agenda electrónica básica.
